Playlist Organizer for Spotify

Last updated: March 20, 2026

Playlist Organizer for Spotify is a browser extension that helps users reorder songs within Spotify playlists.

Information this extension accesses

  • Spotify OAuth access tokens and refresh tokens.
  • Spotify playlist identifiers, track identifiers, and track names.
  • Basic Spotify account session state needed for authentication.

How information is used

  • To sign the user in with Spotify.
  • To refresh Spotify sessions when needed.
  • To reorder tracks within playlists at the user's request.
  • To display extension state inside the popup.

Storage

Spotify access tokens, refresh tokens, and expiration timestamps are stored locally in the browser using Chrome extension storage. They are used only to provide the extension's core functionality.

Sharing

This extension does not sell user data and does not share user data with third parties for advertising or analytics. Data is only sent to Spotify services when required to authenticate the user or perform playlist reorder requests initiated by the user.

Website content

The extension reads limited content from the Spotify web player, such as playlist identifiers, track identifiers, and track names, only to provide reorder controls and carry out the requested action.
