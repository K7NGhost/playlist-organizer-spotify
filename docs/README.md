# Website

This folder contains a simple static site for GitHub Pages:

- `index.html` - landing page for the extension
- `privacy.html` - privacy policy page
- `styles.css` - shared styling

Before publishing:

1. Replace the placeholder support/contact information in `privacy.html`.
2. Replace the GitHub/source link in `index.html` if needed.
3. Publish this folder through GitHub Pages or copy its contents into your Pages branch/root.
